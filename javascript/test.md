# JavaScript 异步编程测验题

## 一、课程说明

1. JavaScript 的执行环境是单线程的，有什么好处和坏处？
2. 为解决上述问题，JavaScript 将任务的执行分为哪两种模式？
3. 异步执行的原理是怎样的？
4. 异步编程的四种方式分别是什么？

## 二、回调函数

1. 回调函数全部都是异步操作吗？
2. 用回调函数实现异步编程有哪些优点和缺点？
3. 什么是回调函数？

## 三、事件监听

1. 事件监听机制实现异步编程，任务执行的顺序取决于什么？
2. 事件监听机制实现异步编程有哪些好处？

## 四、发布/订阅

1. 发布/订阅模式又称为什么？
2. 给数组的末尾添加元素的方法是什么？
3. 删除数组末尾元素的方法是什么
4. 观察者模式实现异步操作的原理是怎样的？

## 五、Promise

1. Promise 有哪三种状态？
2. 上述三种状态有哪些转换途径？
3. Promise 实现异步操作的原理是怎样的？
4. 如何构造 Promise 对象？

## 六、异步编程流程控制

1. 对异步操作的流程控制有哪几种？
2. 对异步操作串行执行的流程控制是如何实现的？
