# node.js 第九次测验题和答案

## 一、web 应用：更轻更快

1. 响应报文的结构是怎样的？
```
状态行
响应头部字段列表
空行
响应主体
```
2. 状态行中包含哪三个信息？
```
HTTP 版本，响应的状态码，响应的状态信息
```
3. 响应头部字段列表以什么形式呈现？
```
键值对
```
4. 状态码有几种？分别表示什么信息？
```
1xx——Informational 基本信息
2xx——Successful 成功信息
3xx——Redirection 重定向
4xx——Client Error 客户端错误
5xx——Server Error 服务器错误
```
5. 请求报文的结构是怎样的？
```
请求行
请求头部字段列表
空行
请求主体
```
6. 请求行中有哪三个信息？
```
请求方法
请求的资源
HTTP 协议版本
```
7. HTTP 协议原生支持的两个基本认证是什么？
```
基本认证
摘要认证
```
8. 从本质上来说 Apache 服务的作用是什么？
```
相当于一个静态网站文件服务程序
```
