# 网页浏览

浏览网页是我们上网时最主要的操作之一，所以我们先从这里开始，保证大家学了之后立马就能用的上。

## 打开网页

### 【o】【记忆：open】
**在当前标签页中**打开一个或多个网址。多个网址间用'urlseparator'分隔（默认为“,”，注意这里逗号后的空格是必须的。）首个网址在当前标签页中打开，其余的则在新标签页中打开。 每个标记按以下顺序进行分析：

- 快速打开网站  
  关键是不用输入网站的完整地址，可以智能匹配 URL，给出匹配列表，在匹配列表中可以用 tab 键快速定位需要打开的目标网站。举例如下：
  - o ho  
  打开 hostedredmine 网站
  - o pr   
  打开 ProcessOn 网站
  - o ji  
  打开简书网站
                
- 快速打开搜索引擎  
  按 o 键后，浏览器底部出现两个窗格，第一个窗格是搜索引擎窗格，第二个窗格是智能补全窗格。这个里面大家一定要把火狐浏览器中，默认添加的很多不用的搜索引擎删除，这样可以保证所有引擎窗格尽可能的少，因为，他会挤占下面窗格的空间。设置方法是【演示一下】。
  
  如果需要快速打开搜索引擎，只要按 tab 键就可以了。按多次 tab 键在搜索引擎之间切换，如果，回退按 shift + tab。如果一路按 tab，就进到智能补全窗格了。
  
  选中搜索引擎之后，按回车键就打开搜索引擎的页面了。按空格键，输入待搜索的关键字，然后按回车键，就打开搜索结果页面了。举例如下：
  - o tab【回车】  
  打开百度页面
  - o tab 王顶【回车】  
  打开百度搜索王顶的结果页面
    
- 快速打开本地文件或文件夹  
  URL 地址可以是本地电脑上的相对或绝对路径的文件名或者路径名。结合 :cd 和 :pwd 命令，在这两个命令的前提下，可以使用相对路径。举例如下：
  - o c:/wd/seIDE/alert/sample.html  
      打开本地 seIDE 仓库中的 sample.html 文件
  - o c:/wd/seIDE/  
    显示本地 seIDE 仓库，浏览文件夹中的内容，利用 hint 模式，快速进入文件夹，打开文件。
  - pwd
    显示当前路径  
  - :cd c:/wd/seIDE/alert
  切换路径到本地 seIDE 仓库的 alert 文件夹
  - sample.html   
  打开本地

### 【O】【记忆：open】
在当前签页打开一个网站其 URL 地址，基于当前网址的增减。注意减的时候按 ctrl + 【backspace】，一次可以删除 URL 中的一个单位。举例如下：
- o git tab tab  
  打开我的 Github 首页
- O ctrl + 【backspace】 + ed  
  打开 Github 软件学院组织首页  

### 【t】【记忆：tab 标签】
t 与 o 类似，但首个网址也在新标签页中打开。

与【t】类似，但最终网页会在新窗口中打开。

### 【W】
基于当前位置打开新窗口。 
  
### 【p】【记忆：paste】
在当前标签页打开当前剪贴板中的网址。结合快捷键【y】使用。

## 导航

### 【gh】【记忆：go home】
 在当前标签页中打开主页。【注意】火狐浏览器的主页设置，about:blank，about:newtab，等常用的特殊地址。

### 【gH】
在新标签页中打开主页。

### 【H】
在浏览历史中后退[count]页。如果给出了[url]，则后退到首个匹配的网址。特别形式:back!则后退到浏览历史中首个页面。

### 【L】
在浏览历史中前进[count]页。如果给出了[url]，则前进到首个匹配的网址。特别形式:forward!则前进到浏览历史中最后一个页面。

### 【gu】【记忆：go up】
转到基于当前目录的第[count]层父目录。

### 【gU】
转到网站根目录。

## 重新载入

### 【r】【记忆：reload】
刷新当前页面。

## 停止

### 【:stop】
停止载入当前页。
 
## 退出

### 【ZQ】
不保存会话而退出。与 :qall 作用相同。

### 【ZZ】
保存会话后退出。无论打开了多少个标签页/窗口，都退出 Vimperator。会话被保存。与 :xall 作用相同。

## 当前目录

跟网页资源管理器结合使用。

### 【~】【记忆：Linux 的主目录一样】
打开主目录。可以结合 hint 模式一块使用，这可能是地球上最快的文件浏览器。

### 【.】【记忆：Linux 的当前路径一样】
打开当前目录。可以结合【:cd】【:pwd】和 hint 模式使用。

### 【:cd】【记忆：change directory】【Linux 命令】
改变当前目录。:cd -改变为前一个访问的目录。

### 【:pwd】【记忆：print working directory】【Linux 命令】 
显示当前目录名。

## 临时禁用 Vimperator

临时禁用 Vimperator 为了让 Vim 用户更愉快地浏览网页，Vimperator 几乎覆盖了 Firefox 所有按键。在某些情况下您可能需要把键击传给 Firefox 或网页，即让这些按键执行原本的功能，此时可使用下列方法：

### 【S-Esc】或【Insert】【记忆：无】  
禁用 Vimperator 所有按键，并把它们传递给下一个事件处理器。在 JavaScript 控制的表单（例如 Gmail 的 RichEdit 表单字段）表现异常或您想使用像 Google Reader 这样带大量快捷键的 web 程序时，这种模式可派上用场。要退出此模式，请再次按【S-Esc】。【Esc】是一种特殊情况，虽然它仍发送给网页，但仍然处理输入字段的失焦。    

### 【i】【记忆：ignore】
如果您只需要传递一个键击给 JavaScript 表单或另一个扩展时，请在该键击前先按 i 键。这种方法同样适用于被 Vimperator 覆盖的其他 Firefox 快捷键，例如：【C-o】。
