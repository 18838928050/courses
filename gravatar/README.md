# Gravatar 全球大头贴
课程以 Redmine 点亮头像为例，说明了 [Gravatar 网站](http://cn.gravatar.com)的使用。因为，Gravatar 是全球大头贴，很多网站都支持，比如 WordPress，Github，等。所以，学会使用这个工具可以一劳永逸。当然，大家要能举一反三，Redmine 网站点亮头像搞明白了，就意味着别的网站点亮头像操作方式类似。
