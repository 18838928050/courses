## 八卦一下 Git 的 logo

这个 logo 为什么张这个样子呢？把图片放平了，很像 Git 三个字母，其实，不是 i 和 t，而是加和减。为什么是加和减呢，颜色已经告诉我们真相了，我们知道文件中删除内容是红色，红色的减号。添加内容是绿色，绿色的加号。我们在 Git Bash 也见到过。可以演示一下。

EGit，为什么叫 EGit 呢？E 代表 Eclipse。EGit，即 Eclipse Git 插件。
