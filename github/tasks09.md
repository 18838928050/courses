# 第九季：Linux Bash 入门

## 任务一：安装实验环境 

要求：
- 启用虚拟化；
- 下载并安装 vmware workstation 12 Pro；
- 新建空白虚拟机；
- 安装 CentOS 最小系统；
- 安装 xshell；
- 安装必要的软件；

## 任务二：新手上路

要求：
- 用 GitHub 账户登录 GitBook；
- 完善 GitBook 上的个人信息；
- 浏览 GitBook 收藏（star）自己感兴趣的电子书；
- 创建空的电子书；

## 任务三：在线制作

要求：  OB
- 在线制作 GitBook 电子书；  
- 电子书的内容随意；  
- 至少包含三个章节；  

## 任务四：本地制作

要求：
- 在本地完善上个任务制作的电子书；  
- 在电子书中增加图片；  
- 在电子书中增加二级目录；  OA

## 任务五：GitHub 集成
